# rehov.net

Personal website — projects, experiments, and writing.

https://www.rehov.net

## Stack

- [Astro v7](https://astro.build) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com) — utility-first styling
- [MDX](https://mdxjs.com) — markdown with component support
- [Firebase Hosting](https://firebase.google.com/products/hosting) — deployment

Content lives in `src/content/projects/` and `src/content/thoughts/` as Markdown or MDX files.

## Commands

| Command             | Action                                      |
| :------------------ | :------------------------------------------ |
| `npm install`       | Install dependencies                        |
| `npm run dev`       | Start dev server at `localhost:4321`        |
| `npm run build`     | Build to `./dist/`                          |
| `npm run preview`   | Preview production build locally            |
| `firebase deploy`   | Deploy to Firebase Hosting                  |
